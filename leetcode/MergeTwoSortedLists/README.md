# Problem

[Merge Two Sorted Lists](https://leetcode.com/problems/merge-two-sorted-lists/)

리스트의 시작노드 `l1, l2` 가 주어진다. 두 리스트를 오름차순으로 병합하라.

# Idea

iterative 하게 접근하면 `O(N) O(1)` 으로 해결할 수 있지만
구현이 복잡하다.

recursive 하게 접근하면 `O(N) O(N)` 으로 해결할 수 있지만
구현이 단순하다.

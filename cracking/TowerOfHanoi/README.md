# Idea

- 기둥이 세개 있다고 디스크가 다섯개 있다고 하자. 첫번째 기둥에서 두번째 기둥으로
  디스크를 4개 옮겨 놓고 첫번째 기둥에서 세번째 기둥으로 한개 옮겨 놓자. 그렇다면
  두번째 기둥의 디스크를 다시 세번째 기둥으로 옮기면 문제는 해결된다.
